# Tradutor-JS

Este é meu primeiro projeto com JS, eu sei que está ruim, não me julgue


## O que é "Tradutor-JS"?

Tradutor-JS é um projeto pessoal com fins de aprendizado para o js, criei um conceito, explorei e coloquei em prática a idéia de fazer um tradutor funcional utilizando HTML, CSS e JavaScript.

## Como fiz este projeto?

Como eu falei, a base do meu projeto é HTML CSS e principalmente o JavaScript, comecei fazendo a base do site utilizando o HTML, coisas como, botões, text inputs, textos, título, organização e outros, foram todos adicionados com HTML.

Logo após criei o layout do site utilizando CSS, coisas como design da tela de fundo, estilização de textos e botões, caixas de escrita etc. Foram todos feitos com CSS.

Logo após estilizar o site com CSS, eu vim com o JavaScript para ser o cérebro do site, ele que faz a mágica acontecer, utilizei alguns códigos "simples" para botões ficarem funcionais, utilizei uma API free para conseguir fazer as traduções, infelizmente pela escolha da API eu tive que fazer algumas alterações no código para garantir que o tradutor funcionasse corretamente.

Então eu comecei a utilizar o JavaScript para fazer os botões serem funcionais e terem uma ação, utilizei JavaScript para conseguir exibir a tradução no local certo, e também utilizei o JavaScript para ser a ponte de comunicação entre, o site, a API, e a máquina! 

## Conclusão

No fim de tudo isso obtive um projeto funcional para o que ele foi destinado, algumas coisas ainda não funcionam como o input do microfone, e a tradução de alguns idiomas, como francês e espanhol, esses idiomas não funcionam a tradução por agora pois eles não foram adicionados na "gambiarra" que fiz para a API conseguir traduzir com êxito, infelizmente pela escolha desta API eu não consegui fazer com que ela detectasse automaticamente o idioma digitado, sendo assim eu tive que manualmente escrever 2 variáveis para a API conseguir se comunicar bem com o JavaScript, variáveis essas que avaliam manualmente se o idioma escrito é pt-BR ou en-US, como eu não adicionei as variáveis do francês e espanhol, infelizmente esses 2 idiomas não podem ser traduzidos, não nesta versão do "Tradutor-JS"
